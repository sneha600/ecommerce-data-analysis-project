# 📊 Marketing Analytics Case Study (E-commerce)

## 🔍 Project Overview
This project focuses on analyzing an E-commerce marketplace dataset to understand customer behavior, product trends, seller distribution, and revenue patterns.

## 📁 Dataset
The dataset includes:
- Customers
- Sellers
- Products
- Orders
- Payments
- Reviews
- Geo-location data

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## 📈 Key Analysis Performed
- Revenue and order trend analysis
- New vs existing customer analysis
- Customer retention insights
- Product and category performance
- Seller and location distribution
- Review and rating analysis

## 💡 Key Insights
- Business growth is mainly driven by new customers
- A few products contribute to repeated purchases
- Seller presence is low compared to number of locations
- Revenue trends show fluctuations (possible seasonality)
- Existing customer contribution is still low

## 🚀 Conclusion
The business shows strong growth potential but can improve by focusing on customer retention, expanding seller base, and maintaining inventory for high-demand products.
